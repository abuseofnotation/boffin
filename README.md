#Boffin Typeface
A simple little typeface for all things technical. A faux monospaced, semi-serif with rounded corners that you will never forget.
![/boffin/promo/01.png]()
The name comes from a British slang word that means "tech-savy person".
![/boffin/promo/03.png]()
Or simply "nerd".
